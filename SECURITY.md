# Security Policy

| Product | Supported here? |
| --- | :---: |
| [Perfect Crewlink (Desktop)](https://github.com/artriy/Perfect-Crewlink) | Yes |
| [Original BetterCrewLink (Desktop)](https://github.com/OhMyGuus/BetterCrewLink) | No |
| [BetterCrewLink (Mobile & Web)](https://github.com/OhMyGuus/BetterCrewlink-mobile) | No |
| [BetterCrewLink (Server)](https://github.com/OhMyGuus/BetterCrewLink-server) | No |
| [CrewLink (PC)](https://github.com/ottomated/CrewLink) | No |
| [CrewLink (Server)](https://github.com/ottomated/CrewLink-server) | No |

Security reports for Perfect Crewlink should be filed against this repository. Legacy upstream
projects should be reported to their respective maintainers.
